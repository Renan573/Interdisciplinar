Linguagem SQL

O que é SQL e para que serve:

SQL (Structured Query Language ou traduzindo para o português: Linguagem Estruturada de Consultas) é uma linguagem comum / padrão para se trabalhar com bancos de dados relacionais. O SQL também é uma linguagem de certa forma fácil, e não é necessita tanto o conhecimento de programação para realizar seus códigos.
Porém o SQL não é apenas uma linguagem, ele é uma linguagem padrão universal usada por programadores e cientistas para manipular bancos de dados por meio dos SGBDs (Sistemas Gerenciadores de Banco de Dados).

O que é SELECT e suas opções (where, order by):

SELECT em SQL é uma declaração que retorna um certo conjunto de resultados de registros de uma ou mais tabelas. Na maioria das aplicações, o SELECT é o comando de Linguagem de Manipulação de Dados mais utilizado.
Junto ao SELECT há cláusulas que podem ajudar a pesquisa de dados nas tabelas, sendo algumas delas a opção WHERE e ORDER BY.

Como usar a cláusula JOIN em SQL:

A cláusula "JOIN" em SQL é uma operação de junção em álgebra relacional, onde combina colunas de uma ou mais tabelas em um único banco de dados relacional.
Existem vários tipos de "JOINS", porém os dois mais usados / principáis são: INNER JOIN e o OUTER JOIN.
O INNER JOIN retorna as linhas (registros) quando ele encontrar pelo menos uma correspondência em ambas das tabelas.
Já o OUTER JOIN, retorna as linhas mesmo quando não houver no mínimo uma correspondência em uma das tabelas. Existem três tipos de OUTER JOINS, sendo o primeiro: LEFT JOIN (onde retorna as linhas da tabela à esquerda), RIGHT JOIN (retorna as linhas da tabela à direita) e o FULL JOIN (onde retorna as linhas quando houver correspondência em qualquer uma das tabelas. Sendo uma combinação do LEFT e RIGHT JOINS).

Para usar a cláusula INNER JOIN é necessário escrever:

SELECT colunas
FROM tabela1
INNER JOIN tabela2
ON tabela1.coluna=tabela2.coluna

Para usar a cláusula LEFT OUTER JOIN é necessário escrever:

SELECT coluna
FROM tabela_esq
LEFT (OUTER) JOIN tabela_dir
ON tabela_esq.coluna=tabela_dir.coluna

Para usar a cláusula RIGHT OUTER JOIN é necessário escrever:

SELECT colunas
FROM tabela_esq
RIGHT (OUTER) JOIN tabela_dir
ON tabela_esq.coluna=tabela_dir.coluna

Para usar a cláusula FULL OUTER JOIN é necessário escrever:

SELECT colunas
FROM tabela1
FULL (OUTER) JOIN tabela2
ON tabela1.coluna=tabela2.coluna
